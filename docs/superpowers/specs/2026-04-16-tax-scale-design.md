## 1. 概述

### 目的
創建一個互動式天秤對比圖，直覺展示三種巴西稅務模式（Lucro Real 正清全報、Lucro Presumido 正清全報、Lucro Presumido 低報）的「總返款中國金額」(A + H)。

### 核心公式
```
總返款中國 = A + H
         H = E - D - I - J
         E = C × 毛利率
         C = CIF + 進口稅費
```

### 三種模式數據
| 模式 | A (CIF) | C (進項) | E (毛利) | F (銷售) | I (IRPJ) | J (CSLL) | D (營運) | H (淨利) | 總返款 A+H |
|------|---------|---------|---------|---------|---------|---------|---------|---------|-----------|
| Lucro Real 正清全報 | 1,000 | 1,500 | 900 | 2,400 | 60 | 36 | 500 | 304 | **1,304** |
| 利潤推算制 正清全報 | 1,000 | 1,500 | 900 | 2,400 | 28.8 | 25.92 | 500 | 345.28 | **1,345.28** |
| 低報 ⚠️ | 1,000 | 750 | 1,387.5 | 2,137.5 | 25.65 | 23.09 | 500 | 338.77 | **1,338.77** |

## 3. 動畫設計

### 天秤搖擺動畫
- **類型**: 持續來回搖擺
- **週期**: 4 秒完成一個週期
- **角度**: 左右各 3 度
- **緩動**: `ease-in-out` 曲線

### 懸停效果
1. 砝碼放大 1.05 倍
2. 天秤向該側傾斜 5 度
3. 該砝碼輕微發光 (box-shadow)

### 點擊展開動畫
1. 點擊的砝碼展開成「公式水晶盒」
2. 其他砝碼縮小並靠邊
3. 展開動畫：scale + opacity，300ms

### 檔案位置

### Props 介面
```typescript
interface TaxMode {
  id: string;
  label: string;
  labelShort: string;
  icon: string;
  status: 'compliant' | 'warning';
  data: {
    A: number;  // CIF 實際貨價
    B: number;  // 申報金額
    importTax: number;  // 進口稅費
    C: number;  // 進項發票
    grossMargin: number;  // 毛利率
    E: number;  // 毛利
    F: number;  // 銷售發票
    I: number;  // IRPJ
    J: number;  // CSLL
    D: number;  // 營運成本
    H: number;  // 稅後淨利
    totalRemittance: number;  // A + H
  };
}

interface Props {
  modes?: TaxMode[];
  class?: string;
}
```

## 5. 互動流程

### 預設狀態
1. 頁面載入
2. 天秤開始緩慢搖擺
3. 三個砝碼顯示基礎訊息（H 值、總返款）

### 懸停流程
1. 滑鼠移入砝碼
2. 砝碼放大 + 發光
3. 天秤傾斜向該側
4. 顯示該模式的簡要說明

### 點擊展開流程
1. 點擊砝碼
2. 展開成「公式水晶盒」
3. 顯示完整公式推導鏈
4. 點擊其他砝碼或收起按鈕可關閉
