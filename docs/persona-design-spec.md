## 🎯 設計理念

### 核心概念：「軍師引路 × 英雄衝關」

```
讀者 = 野心勃勃的探險家
軍師 = AI 引導者（對話式診斷）
手冊 = 英雄之旅地圖
閱讀 = 衝關挑戰
完成 = 金色印章 + 自我提升的驕傲
```

### 用戶旅程

```
進入首頁
  ↓
🧭 看到巨型互動羅盤（4 方位發光）
  ↓
🎭 點擊任意方位 → 軍師現身
  ↓
❓ 回答 3 個診斷問題（身份 → 目標 → 進度）
  ↓
📜 軍師揭示個人化戰略路徑（卷軸展開動畫）
  ↓
🗺️ 點擊「踏上征途」→ 進入第一篇文章
  ↓
📖 閱讀文章 → 完成互動組件 → 獲得金色印章
  ↓
➡️ 點擊「下一關」→ 直接跳到下一篇文章
  ↓
🔁 重複直到終點 → 🏆 征服巴西！
```

**關鍵原則**：讀者一旦選擇路徑，就**不需要再回到首頁**。所有導航都在文章頁內完成。

## 🧩 四大階段

### 階段一：互動羅盤（首頁入口）

**視覺**：巨型 SVG 羅盤，4 個方位用品牌色漸變
- 🔵 北：偵察兵（Preparation）— 鉑金 #E8E4D9
- 🟡 東：奠基者（Foundation）— 金色 #D4A843
- 🟢 南：領航員（Operations）— 霓虹綠 #00FF87
- 🟣 西：收割者（Harvest）— 霓虹黃 #E5FF00

**互動**：hover 時方位發光，點擊後觸發軍師動畫

### 階段二：軍師診斷（3 問）

**Q1：你的身份？**
- 🧑‍💼 個人投資者 → 走簽證路線
- 🏢 企業派出 → 走企業路線
- 🌐 跨境賣家 → 純電商路線

**Q2：你的終極目標？**
- 💰 利潤匯回 → 賺巴西錢，匯回母國
- 🏠 移民定居 → 拿身份，在巴西生活
- 📦 規模擴張 → 巴西作拉美樞紐
- 🎯 快速試水 → 最小成本測試

**Q3：目前進度？**
- 🔵 零基礎 → 還在了解階段
- 🟡 準備中 → 已決定，正在籌備
- 🟢 已落地 → 公司/簽證已搞定
- 🟣 運營中 → 已有銷售，需優化

### 階段三：戰略卷軸揭示

根據 3 問答案，匹配 6 條路徑之一：

| 路徑 | 名稱 | 關卡 | 時間 | 適合 |
|------|------|------|------|------|
| A | 🛡️ 移民征途 | 19 | ~8h | 個人 + 移民 + 零基礎 |
| B | 🚀 閃電出海 | 5 | ~2.5h | 跨境賣家 + 試水 + 零基礎 |
| C | 🏗️ 企業遠征 | 17 | ~6h | 企業 + 擴張 + 準備中 |
| D | ⚡ 落地加速 | 8 | ~4h | 已落地 + 利潤匯回 |
| E | 🔧 運營優化 | 5 | ~2.5h | 運營中 + 利潤匯回 |
| F | 🎯 試水偵察 | 3 | ~1.5h | 跨境賣家 + 試水 + 零基礎 |

### 階段四：英雄之旅地圖展開

點擊「踏上征途」→ 跳轉到第一篇文章 → 文章頁內：
- **頂部**：小地圖組件（sticky，顯示進度和所有關卡狀態）
- **底部**：下一關導航組件（完成閱讀後淡入，有慶祝動畫）
- **不需要回到首頁**

## 📐 組件設計

### 1. 頂部小地圖（MiniMap）
- **位置**：`position: sticky; top: 56px;`
- **內容**：路徑名稱、關卡進度、當前標題、預估時間、節點軌道
- **節點**：圓形，4 種狀態（completed/current/available/locked）
- **互動**：可點擊已完成的節點複習，可點擊 available 節點跳轉

### 2. 底部導航（BottomNav）
- **位置**：文章內容結束後
- **觸發**：滾動到底部時淡入
- **內容**：
  - 慶祝動畫（✅ 關卡 X/Y 已完成 + 🏅 金色印章）
  - 上一關/下一關鏈接
  - 主要 CTA：「🔥 繼續征途 → [下一章標題]」
  - 輔助操作：本章詞彙、收藏此關、查看完整地圖

### 3. 首頁診斷表單
- **位置**：替換現有 Hero 區域
- **形式**：3 步問答，每步卡片式選擇
- **結果**：卷軸展開動畫，顯示路徑名稱、關卡數、預估時間、閱讀清單
- **CTA**：「🔥 踏上征途」→ 直接跳到第一篇文章

- **動畫**：CSS animations + IntersectionObserver
